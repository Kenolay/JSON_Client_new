16. Сетевая игра «Бойцы» для двух игроков. У каждого
игрока есть 8 бойцов, различной силы, измеряющейся целыми
числами от 1 до 8. Игроки в тайне друг от друга расставляют
бойцов в желаемой последовательности. Когда бойцы разных
игроков оказываются друг напротив друга, то боец с меньшей
силой уничтожается, и соответствующий игрок зарабатывает 1
очко. Если в такой ситуации силы бойцов одинаковые, то
уничтожаются оба бойца, и ни один из игроков не получает
очков. Выигрывает игрок, который набрал больше очков, чем
другой игрок.

Программа клиент, сервер находится на https://github.com/Kenolay/JSON_Server_new/blob/master/README.md
